Carnet - это мощное приложение для создания заметок, цель которого не только в написании вашего 
списка покупок, но и в написании более длинных текстов, рассказов и т. д. 

Carnet - это открытый исходный код, Carnet не отслеживает вас, Carnet не содержит рекламы, Carnet бесплатен. Мы лишь рассчитываем на пожертвования :) 

Carnet доступен для Android и Linux, а также в вашем браузере с полной совместимостью с Windows, 
Mac и Linux, с возможностью синхронизации (не обязательно) 

'' 'Особенности' '' 

* Полноценный редактор: полужирный / курсив / подчеркивание / цвет / выделение 
* Импорт из Google Keep (только на компьютере, затем синхронизация на мобильном устройстве) 
* Вставка изображений / обзор 
* Открытый HTML формат
* Организация по папкам, выбор корневой папки хранения в соответствии с вашими потребностями 
* Ключевые слова
* Быстрый просмотр ваших последних заметок 
* Поиск суммы ваших заметок 
* Защита приложения с помощью пин-кода (заметки не шифруются) на Android 
* Статистика: слова / предложения / символы 
* Синхронизация с NextCloud 
* Запись звука 
* Онлайн-редактор в качестве приложения NextCloud 
* Темная тема 
* Изменение цвета фона заметки 
* Список дел 

'' 'В разработке' ''

* Более легкая версия для Linux без Electron
