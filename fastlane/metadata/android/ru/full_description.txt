Carnet - это мощное приложение для создания заметок, предназначенное не только для составления Ваших 
списков покупок, но и более объемных текстов, сочинений и т.п. 

Carnet – бесплатное ПО с открытым исходным кодом. Carnet не следит за Вами и не содержит рекламы. Мы лишь надеемся на пожертвования :) 

Carnet доступен на Android и Linux, а также в Вашем браузере, обеспечивая полную совместимость с Windows, 
Mac и Linux с возможностью синхронизации (не обязательна) 

'''Особенности''' 

* Полноценный редактор: полужирный / курсив / подчеркивание / цвет / выделение 
* Импорт из Google Keep (только на компьютере и последующей синхронизацией с мобильным устройством) 
* Вставка изображений и присвоение рейтинга
* Открытый HTML формат
* Организация по папкам и выбор корневого каталога хранения по Вашему усмотрению
* Ключевые слова
* Быстрый просмотр последних записей 
* Поиск среди Ваших заметок
* Защита приложения с помощью пин-кода на Android (заметки не шифруются) 
* Ведение статистики: слова / предложения / символы 
* Синхронизация с NextCloud / OwnCloud
* Запись звука
* Онлайн-редактор в качестве NextCloud App
* Темная тема 
* Изменение цвета фона заметки 
* Списки задач
* Оповещения

'''В разработке''' 

* Улучшения Perf 
* Оптимизированная версия для Linux без Electron 
* Приложение для Windows / macOS 
* Многие вещи, о которых я ещё не думал
