Carnet est une application de prise de note puissante qui permet certes d'écrire votre liste de course mais peut aussi être utilisée pour écrire de courts ou longs textes, histoires, coller des images, comme un véritable Carnet.

Carnet est Open Source, Carnet ne vous espionne pas, Carnet ne contient aucune publicité, Carnet est gratuit. Nous comptons sur vos dons :)

Carnet est disponible sur Android et Linux ainsi que dans votre navigateur internet (Mac, Windows et Linux) si vous choisissez d'activer la synchronisation.

'''Fonctionnalités'''

* Éditeur complet : Gras/Italique/Souligné/Couleur/Surlignage/Alignement
* Import depuis Google Keep (seulement sur la version pour ordinateur, qui ensuite peut se synchroniser avec l'application mobile)
* Organisation par dossier avec choix du dossier de stockage de vos notes
* Mots-clefs / Labels
* Navigation rapide parmi vos dernières notes
* Recherche parmi vos notes
* Protection par code pin (sans chiffrement)
* Statistiques : compter vos mots / phrases / caractères
* Synchronisation avec NextCloud
* Enregistrement audio
* Disponible sur le magasin d'application NextCloud
* Thème sombre
* Choix de la couleur de fond de la note
* Todo-Lists
* Rappels
